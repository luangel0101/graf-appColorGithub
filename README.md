# Introducción a la Programación de Interfaces Web

## Descripción de la aplicación

Esta aplicación web fue desarrollada como parte de la actividad 1.1 de la materia
Introducción a la programación de interfaces Web.

La aplicación permite al usuario seleccionar un color mediante un **color picker**
y visualizar el color elegido tanto en un cuadro de muestra como en los controles
individuales de los valores RGB (rojo, verde y azul).

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidad principal

- Selección de color usando un input tipo color
- Visualización del color seleccionado en tiempo real
- Actualización automática de los valores RGB
- Diseño sencillo, moderno y fácil de usar
